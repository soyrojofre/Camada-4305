Git

  1. Entrar a la consola y pararnos en la carpeta que queremos trabajar.
  2. Si todavia no es un repositorio, ejecutar git init. Si ya es un repositorio, ir al paso 3.
  3. Trabajar en mis archivos normalmente, y cada tanto, ejecutar un git add . para agregar los archivos modificados al repositorio
  4. Una vez que tengo una version copada, ejecuto git commit -m "mensaje".
  5. Volver al paso 3.

GitHub

  1. Ir a GitHub y crear un repositorio.
  2. Copiar y ejecutar git remote add origin...
  3. Ejecutar el git push -u origin master
  4. Volver al paso 3 de la otra lisra. Y cuando quiera volver a subir, simplemente hago el paso 2 de esta (git push)
